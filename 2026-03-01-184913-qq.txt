  使用方法

  1. 获取 QQ 邮箱授权码：
  - 登录 QQ 邮箱网页版 (mail.qq.com)
  - 点击"设置" → "账户"
  - 开启"IMAP/SMTP 服务"
  - 生成授权码（16 位）

  2. 代码示例：

  from qq_email_validator import QQEmailClient, send_qq_email

  # 创建客户端
  client = QQEmailClient(
      qq_email="your_qq@qq.com",
      auth_code="your_auth_code"  # 16 位授权码
  )

  # 发送邮件
  client.send_email(
      to_address="recipient@example.com",
      subject="测试邮件",
      content="这是邮件内容",
      sender_name="张三"
  )

  # 收取未读邮件
  emails = client.fetch_emails(unread_only=True, limit=10)

  # 验证邮件内容（检查手机号、身份证、姓名）
  results = client.validate_all_unread()

  配置说明

  - IMAP 服务器: imap.qq.com:993
  - SMTP 服务器: smtp.qq.com:465
  - 认证方式: 授权码（不是 QQ 密码）
